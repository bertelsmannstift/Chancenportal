# Changelog

## Release 2020-02-27

### Release für TYPO3 9.5
Mit diesem Release wurde die zugrundeliegende TYPO3-Version von TYPO3 8.7 LTS auf TYPO3 9.5 LTS geändert. TYPO3 8.7 LTS wird mit diesem Release nicht mehr unterstützt.
